## Purpose

This folder records how the Builder Assistant may use the TUYA workbook.

The workbook is a methodology reference for Elementor V4 learning, not a runtime authority over current UI controls.

---

## What It Helps With

```text
Element Tree thinking
Local Class vs Global Class decisions
Variables vs Classes vs Components
Box Model and sizing decisions
Flex/Grid layout reasoning
Responsive workflow
Position, z-index and overflow reasoning
RTL and reading order
State and focus checks
Performance and structure audit
```

---

## Priority Boundary

```text
Official Elementor documentation > current UI/version evidence > Builder_Context_Package > workbook > case memory > model inference
```

For executable controls, current UI evidence remains required.
