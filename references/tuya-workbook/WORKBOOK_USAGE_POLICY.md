# WORKBOOK_USAGE_POLICY

Version: 0.1.0
Status: active

---

## Role

The TUYA workbook is a learning and methodology reference.

It supports explanations, safe patterns, responsive thinking, and design-system decisions.

---

## Priority

```text
Official Elementor docs
Current UI/version evidence
Builder_Context_Package
Workbook
Case memory
Model inference
```

For exact UI actions, current UI/version evidence is required.

---

## Good Uses

```text
concept explanation
pattern support
risk reminders
responsive workflow reminders
design-system framing
```

---

## Weak Uses

Workbook evidence alone is weak for:

```text
current control existence
exact panel path
installed-version support
exact numeric values
project-specific class names
production status
```
