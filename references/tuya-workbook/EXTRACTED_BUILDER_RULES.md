Version: 0.1.0

---

## Rules

```text
Structure before styling.
Meaningful content stays in normal flow.
Visual overlay stays in a bounded stage.
Verify one repeated item before duplication.
Classify reuse before choosing Variable, Class, Component, or CSS.
Visual estimates are provisional values.
Desktop stability comes before responsive tuning unless user requests otherwise.
Frontend evidence checks spacing and clipping.
Editor evidence checks current controls, active class, and Structure Panel state.
Decoration-only layers stay separate from meaningful content.
Unknowns remain unresolved until evidence resolves them.
```
