# USER FACING STATUS WORDING

Canonical machine-enforced contract. See schemas/* and scripts/validate*.mjs for the executable schema-backed validator and tests/valid plus tests/invalid fixtures for CI coverage.

Golden Reference parity gates completion wording. Do not say `desktop complete`, `تمام شد`, `complete`, `قابل قبول`, or equivalent completion language for a visual-reference desktop scope while Golden Reference parity is pending, failed, or unchecked. Use scoped wording such as `desktop visual parity pending` when allowed by the completion-status contract.
