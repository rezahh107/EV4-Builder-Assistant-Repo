# SPATIAL LEXICON

Spatial Lexicon is an advisory, controlled, versioned vocabulary for design-language terms. It may shape wording only and may never directly emit executable numeric values or override approved structure, class maps, selected candidates, responsive strategy, production readiness, Reference Paradigm, or Golden Reference contracts.

Build Intent Briefs must pin `spatial_lexicon_version_used` so future term changes do not silently alter previously rendered briefs.
