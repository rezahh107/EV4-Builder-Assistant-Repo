# BUILD INTENT BRIEF

Build Intent Brief is explanatory only. Builder renders deterministic Persian mental-model text from an upstream validated template registry and structured tokens; Builder does not generate, rewrite, paraphrase, or infer design narrative.

A brief is displayable only when schema validation passes, pinned template/registry/lexicon/golden-reference versions are present, deterministic render output equals `rendered_text`, and `rendered_brief_hash` matches. It may not override approved structure tree, approved class map, selected_candidate_id, reference paradigm lock, Golden Reference Contract, or production_ready.
